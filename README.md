# SCT_DA_4 - Business Insights Report (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on a Marketing Campaign dataset to understand customer behavior, purchasing patterns, and campaign effectiveness.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Marketing Campaign Dataset containing customer demographics, spending habits, and campaign response information.

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Duplicate Value Check
- Age Distribution Analysis
- Education Distribution Analysis
- Marital Status Analysis
- Product Spending Analysis
- Purchase Channel Analysis
- Correlation Analysis
- Campaign Response Analysis

## Key Insights
- Middle-aged customers form the majority of customers.
- Married customers represent the largest customer segment.
- Wine and Meat products contribute the highest spending.
- Store purchases are higher than Web and Catalog purchases.
- Campaign response rates are relatively low.

## Recommendations
- Focus marketing efforts on high-spending customer segments.
- Increase investment in high-performing product categories.
- Improve digital marketing strategies to boost online purchases.
- Create targeted campaigns to improve response rates.
